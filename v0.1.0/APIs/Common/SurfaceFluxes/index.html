Surface Fluxes

Surface Fluxes

ClimateMachine.SurfaceFluxes — Module
SurfaceFluxes

Surface flux functions, e.g., for buoyancy flux, friction velocity, and exchange coefficients.

Sub-modules

  • module Byun1990
  • module Nishizawa2018

Interface

  • compute_buoyancy_flux computes the buoyancy flux
  • In addition, each sub-module has the following functions:
    • monin_obukhov_len computes the Monin-Obukhov length
    • compute_friction_velocity computes the friction velocity
    • compute_exchange_coefficients computes the exchange coefficients

References

ClimateMachine.SurfaceFluxes.Nishizawa2018.compute_friction_velocity — Function
compute_friction_velocity(param_set, u_ave, θ, flux, Δz, z_0, a, Ψ_m_tol, tol_abs, iter_max)

Computes friction velocity, in Eq. 12 in, by solving the non-linear equation:

u_ave = ustar/κ * ( ln(Δz/z_0) - Ψ_m(Δz/L) + z_0/Δz * Ψ_m(z_0/L) + R_z0 [ψ_m(z_0/L) - 1] )

where L is a non-linear function of ustar (see monin_obukhov_len).
